## 简介

### 什么是这个Skill？

这是一个**专业的竞品分析框架和工具集**，帮助你系统地研究和分析竞争对手。

它不仅提供了理论框架，还包括：
- 📊 **10大分析维度** - 从市场、产品、数据、运营多个角度分析
- 🛠️ **自动化工具** - Python脚本支持数据收集和表格生成
- 📈 **数据对标表** - 即用的Excel模板
- 📝 **案例和指南** - 实战案例和最佳实践
- 🔄 **持续更新** - 支持长期的竞品跟踪

### 核心特点

✨ **全面** - 10大维度的完整分析框架  
🎯 **实用** - 直接生成可执行的建议  
🔗 **可集成** - 支持多个AI平台的API调用  
📊 **数据驱动** - 强调数据验证和来源标注  
🚀 **易上手** - 交互式流程，逐步引导

## 快速开始

### 最快上手（5分钟）

```bash
# 1. 进入Skill目录
cd competitive-analysis-skill

# 2. 运行初始化脚本
python scripts/initialize_analysis.py

# 3. 按照提示输入信息
- 你的品牌名称（可选）
- 要分析的竞品（3-5个）
- 分析重点维度
- 交付形式

# 4. 获得分析计划
输出：analysis_plan_[时间].json
```

### 逐步使用（1-2周）

**第1天：启动项目**
```
□ 明确分析目的（融资/产品/战略等）
□ 选择3-5个核心竞品
□ 定义分析重点维度
□ 生成数据收集清单
```

**第2-4天：收集数据**
```
□ 竞品基础信息（融资、创始人、定位）

□ 功能清单和产品体验
□ 运营和营销信息
```

**第5-6天：深度分析**
```
□ 功能和产品深度分析
□ 商业模式和盈利分析
□ SWOT分析
□ 机会和威胁评估
```

**第7天：报告生成**
```
□ 整理所有数据和分析
□ 生成对标表和对比图
□ 撰写最终报告
□ 提出战略建议
```

### API集成（高级）

如果你想在自己的应用中集成：

```python
import anthropic

client = anthropic.Anthropic()

# 读取SKILL文档
with open('SKILL.md', 'r', encoding='utf-8') as f:
    skill_content = f.read()

# 构建提示词
prompt = f"""
你是一个竞品分析专家。
以下是竞品分析框架：

{skill_content}

请使用这个框架，为以下竞品进行深度分析：
{user_competitors}

分析目标：{user_goal}
重点维度：{user_focus}

请生成完整的竞品分析报告。
"""

# 调用API
message = client.messages.create(
    model="claude-opus-4-20250805",
    max_tokens=4000,
    messages=[
        {"role": "user", "content": prompt}
    ]
)

# 获取结果
analysis_result = message.content[0].text
print(analysis_result)
```

## FAQ

### Q: 我应该分析多少个竞品？

**A:** 建议3-5个。
- **3个**：最小化分析，快速了解市场
- **5个**：标准配置，全面了解竞争格局
- **5个+**：深度分析，需要更多时间

### Q: 如果数据找不到怎么办？

**A:** 标记为"待收集"或"N/A"，继续。
- 优先关注关键数据（融资、定位、用户规模）
- 次要数据可以后续补充
- 不要为了数据而停滞

### Q: 竞品分析需要多长时间？

**A:** 取决于深度和数据可获得性。
- **快速版**（5-7天）：重点维度分析
- **标准版**（1-2周）：完整10大维度
- **深度版**（2-4周）：包含用户调研和行业分析

### Q: 能否在线实时做分析？

## 最后的话

竞品分析的目的不是模仿，而是：
1. **理解市场** - 了解竞争环境和用户需求
2. **找到差异** - 识别市场空白和机会
3. **制定战略** - 为自己的产品找到独特位置
4. **快速成长** - 学习竞品的优秀实践，避免它们的错误

最好的竞品分析应该这样结束：

> "虽然竞品X在某些方面做得很强，
> 但用户仍有以下未被满足的需求...
> 而我们的独特优势恰好能满足这些需求。"
