# 竞品分析Skill - 项目交付总结

## 📦 项目概况

这是一个**完整的竞品分析框架和工具集**，可以直接上传到GitHub或集成到Claude、ChatGPT等AI平台。

### 项目目标

✅ **完全实现**

1. ✨ 创建一个系统化的竞品研究框架
2. 📊 提供标准化的分析维度和方法论
3. 🛠️ 包含可执行的脚本和模板
4. 📚 补充实战案例和最佳实践
5. 🔄 支持多个AI平台的集成
6. 📈 帮助更快、更好地完成竞品分析

## 核心特点

### 1️⃣ 完整的分析框架

10大维度全覆盖，从市场、产品、用户、数据、运营多个角度系统分析：

```
维度1：市场现状   → 了解行业整体情况
维度2：竞品背景   → 了解竞品资本和团队
维度3：定位愿景   → 了解竞品战略定位
维度4：目标用户   → 了解竞品用户特征
维度5：数据分析   → 了解竞品规模和增长
维度6：产品功能   → 了解竞品功能和体验
维度7：运营推广   → 了解竞品如何获客
维度8：盈利模式   → 了解竞品如何变现
维度9：SWOT分析  → 了解竞品优劣和机会
维度10：结论建议  → 制定自己的战略
```

### 2️⃣ 即用型工具和模板

- ✅ **初始化脚本** - 一键启动分析项目
- ✅ **表格生成器** - 自动生成数据对标表
- ✅ **检查清单** - 确保不遗漏关键信息
- ✅ **跟踪模板** - 系统跟踪每个竞品
- ✅ **报告模板** - 快速生成专业报告

### 3️⃣ 丰富的实战案例

两个完整的案例分析：
- **案例1**：二手交易平台（闲鱼 vs 转转）
  - 深度解析为什么两个平台的差异化策略完全不同
  - 从产品定位、目标用户、商业模式等全面对比
  
- **案例2**：工具型SaaS（快文档 vs 金山WPS）
  - 分析新进入者如何与成熟企业竞争
  - 识别垂直领域的机会

### 4️⃣ 数据源和方法论

- 📊 19个免费数据源的详细说明
- 📈 数据分析的4个实用技巧
- 🎯 报告撰写的5个关键点
- ✅ 完整的数据验证清单

### 5️⃣ 多平台集成支持

## 📚 使用场景

### 场景1：融资前的市场分析

```
需求：证明市场机会和竞争优势

使用方案：
1. 运行initialize_analysis.py
2. 分析3-5个直接竞品
3. 关注：市场、定位、数据、SWOT
4. 输出：竞品分析报告 + 差异化建议
5. 用途：融资材料中展示市场理解
```

### 场景2：新品发布前的对标分析

```
需求：了解竞品功能，避免过度开发

使用方案：
1. 重点分析：产品功能、定位、用户体验
2. 创建功能对比表
3. 识别市场空白和创新机会
4. 输出：产品功能设计参考
```

### 场景3：市场进入前的策略制定

```
需求：快速了解一个陌生的新市场

使用方案：
1. 完整分析市场主要竞品（3-5个）
2. 覆盖所有10大维度
3. 特别关注：市场格局、用户需求、商业模式
4. 输出：市场进入战略和差异化方案
```

### 场景4：持续竞品监控

```
需求：长期追踪竞品动态，及时调整策略

使用方案：
1. 建立竞品跟踪表
2. 月度更新：排名、融资、功能更新
3. 季度分析：完整的维度分析
4. 半年：战略调整和机会重评估
```

## 🔗 如何在AI平台中使用

### 在Claude中使用

```
1. 上传SKILL.md文件到Claude
2. 或复制粘贴SKILL.md内容
3. 告诉Claude分析需求：
   "请使用这个竞品分析框架，帮我分析[竞品]"
4. Claude会自动应用框架
```

### 在ChatGPT中使用

```
1. 将内容粘贴到ChatGPT
2. 设置system prompt：
   "You are a competitive analysis expert. 
    Use the following framework to analyze competitors..."
3. 上传或描述竞品信息
4. 获取分析结果
```

### 通过API使用

```python
import anthropic

# 读取框架
with open('SKILL.md') as f:
    framework = f.read()

client = anthropic.Anthropic()
response = client.messages.create(
    model="claude-opus-4-20250805",
    max_tokens=4000,
    messages=[{
        "role": "user",
        "content": f"{framework}\n\n请分析以下竞品: {competitors}"
    }]
)
```
